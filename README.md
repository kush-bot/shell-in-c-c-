# shell-in-c-c-
A basic shell written in c++ on UNIX operating system

![GitHub Logo](https://github.com/kush-bot/shell-in-cpp/blob/main/Screenshot_2024-02-03_16_52_39.png)

- Basic command execution
- Command-line parsing

## Getting Started
